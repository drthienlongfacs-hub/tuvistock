## I. CÁT CÁCH (Cách cục tốt)

### 1. PHỦ LỘC ĐỒNG CUNG — "Tích Ngọc Đôi Kim"

| Hạng mục | Nội dung |
|---|---|
| **Tên** | Phủ Lộc đồng cung / Tích ngọc đôi kim |
| **Điều kiện** | Thiên Phủ + Lộc Tồn đồng cung tại Mệnh, Tài hoặc Điền |
| **Câu phú** | *"Lộc Tồn thủ ư Tài Trạch, tích ngọc đôi kim"* — *"Phủ Lộc đồng cung, cự vạn chi tư"* |
| **Nguồn** | kabala.vn, mogi.vn, lyso.vn |
| **Ý nghĩa** | 2 Tài Tinh ngồi cùng kho = Tích lũy vàng ngọc, tài sản cỡ vạn. Đại phú |
| **Trên lá số** | ✅ **Thỏa mãn.** Thiên Phủ (Đ) + Lộc Tồn (Đ) đồng cung tại **Mệnh Hợi** |
| **Mức tác động** | Cát **9/10** — Nhưng bị Tuần Không ức chế tiền vận → Phát muộn |
| **Cảnh báo** | *"Phủ phùng Không diệu, bất quý"* (tuviglobal.com). Tuần làm giảm hiệu lực |

### 2. TAM KỲ GIA HỘI (Biến thể — Lộc Quyền Khoa hội Tam Phương)

| Hạng mục | Nội dung |
|---|---|
| **Tên** | Tam Kỳ Gia Hội |
| **Điều kiện** | Hóa Lộc + Hóa Quyền + Hóa Khoa hội tại Mệnh hoặc Tam phương tứ chính |
| **Câu phú** | *"Ba kỳ vây hướng Tử Vi cung... Công danh phú quý bất lôi đồng"* |
| **Nguồn** | kabala.vn, phongthuythanglong.vn |
| **Ý nghĩa** | Lộc (Tài) + Quyền (Lực) + Khoa (Danh) = Phú quý song toàn, chân tể tướng |
| **Trên lá số** | ✅ **Thỏa mãn (biến thể).** Hóa Khoa tại **Mệnh Hợi**. Hóa Quyền tại **Di Tỵ** (xung chiếu). Hóa Lộc tại **Nô Thìn** (không trực tiếp trong tam phương Mệnh nhưng rọi thông qua hệ thống) |
| **Mức tác động** | Cát **7/10** — Biến thể (Lộc không ở tam phương trực tiếp). Nhưng Khoa + Quyền hội chiếu Mệnh-Di = Rất mạnh |
| **Cảnh báo** | Phú: *"Tam Kỳ kỵ gặp Không Kiếp, Kình Đà"*. Lá số có Kình Đà giáp + Không Kiếp giáp → Giảm hiệu lực thêm |

### 3. LỘC MÃ GIAO TRÌ

| Hạng mục | Nội dung |
|---|---|
| **Tên** | Lộc Mã Giao Trì |
| **Điều kiện** | Lộc Tồn (hoặc Hóa Lộc) + Thiên Mã hội hợp tại tam phương tứ chính |
| **Câu phú** | *"Lộc Mã giao trì, phát tài viễn quận"* — *"Lộc Mã đồng cung, hào phú gia tư"* |
| **Nguồn** | thayungkhiem.vn, huyenhocvietnam.vn, tuviglobal.com |
| **Ý nghĩa** | Ngựa chở Lộc = Phát tài ở xa quê, kiếm tiền qua di chuyển/BĐS/kinh doanh xa |
| **Trên lá số** | ✅ **Thỏa mãn (Nhị hợp).** Lộc Tồn tại **Mệnh Hợi** nhị hợp Thiên Mã (Đ) tại **Điền Dần**. Tuy không đồng cung, nhị hợp là liên kết rất chặt |
| **Mức tác động** | Cát **7/10** — Nhị hợp truyền ngầm (10% kênh) nhưng sao đắc nên hiệu quả cao. Giảm 1 bậc so với cùng sao nếu chiếu trực tiếp |

### 4. NHẬT NGUYỆT GIÁP (MỆNH/THÂN)

| Hạng mục | Nội dung |
|---|---|
| **Tên** | Nhật Nguyệt Giáp Mệnh / Nhật Nguyệt Giáp Thân |
| **Điều kiện** | Thái Dương và Thái Âm Miếu/Vượng nằm ở 2 cung kề cạnh giáp 1 cung |
| **Câu phú** | *"Nhật Nguyệt giáp Mệnh (hoặc Tài), nhất thế vinh hoa"* — *"Nhật Nguyệt giáp, bất quý tắc phú"* |
| **Nguồn** | kabala.vn, tracuutuvi.com, xemtuong.net, lyso.vn |
| **Ý nghĩa** | Mặt trời + Mặt trăng sáng tỏ kẹp hai bên = Phú quý vang danh |
| **Trên lá số** | ✅ **Thỏa mãn cho Cung Thân (Sửu).** Thái Dương (V) tại Dần + Thái Âm (V) tại Tý giáp Sửu (Thân). Đặc biệt: *"Cung Sửu có Vũ Tham tọa thủ mà Nhật Nguyệt giáp → Đẹp nhất"* (lyso.vn) |
| **Mức tác động** | Cát **8/10** — Giảm vì Nhật ngộ Triệt (Dần) và Âm ngộ Kiếp Kình (Tý) |
| **Cảnh báo** | Phú: *"Nhật Nguyệt giáp mà Kình Đà xung hãm, chỉ tu hành"* (kabala.vn). Cần kiểm soát |

### 5. VŨ THAM ĐỒNG HÀNH TẠI SỬU MÙI

| Hạng mục | Nội dung |
|---|---|
| **Tên** | Vũ Tham Đồng Hành (Chính Cách) |
| **Điều kiện** | Vũ Khúc + Tham Lang đồng cung tại Sửu hoặc Mùi (Tứ Mộ) |
| **Câu phú** | *"Tham Vũ đồng hành, vãn niên y lộc noãn phong dinh"* — *"Tham Vũ đồng hành bất phát thiếu niên nhân"* — *"Tham Vũ đồng cung, uy danh lừng lẫy biên cương"* |
| **Nguồn** | tuvivietnam.vn, xemngay.com, tuvisaigon.vn, lyso.vn |
| **Ý nghĩa** | Tiền bần hậu phú. Tuổi trẻ vất vả, sau 30-40 phát rực rỡ. Uy danh, phú quý |
| **Trên lá số** | ✅ **Thỏa mãn.** Vũ Khúc (M) + Tham Lang (M) đồng cung tại **Phúc/Thân Sửu** |
| **Mức tác động** | Cát **9/10** — Chính Cách, cả hai đều Miếu địa |

### 6. THAM LINH TỊNH THỦ (Biến thể — Tam hợp)

| Hạng mục | Nội dung |
|---|---|
| **Tên** | Tham Linh Tịnh Thủ |
| **Điều kiện** | Tham Lang + Linh Tinh đồng cung hoặc tam hợp. Tốt nhất tại Tứ Mộ |
| **Câu phú** | *"Tham Linh tịnh thủ, tướng tướng chi danh"* — *"Tham Lang cánh đắc dữ Linh phùng, nhập miếu cung trung phúc khí long"* |
| **Nguồn** | tuvisaigon.vn, lyso.vn, tuancaopro.com |
| **Ý nghĩa** | Bạo phát tài lộc bất ngờ. Anh hùng cầm quân. Kinh doanh đại thắng |
| **Trên lá số** | ✅ **Thỏa mãn (biến thể Tam hợp).** Tham Lang (M) tại Sửu — Linh Tinh (H) tại Dậu (Thê). Tam hợp Sửu-Tỵ-Dậu. Linh hãm → Giảm hiệu lực |
| **Mức tác động** | Cát **6/10** — Linh hãm nên bạo phát kém bền, cần kiểm soát |
| **Cảnh báo** | Phú: *"Linh hãm, phát nhanh tàn nhanh"*. Cần thêm cát tinh ổn định |

| Hạng mục | Nội dung |
|---|---|

| **Trên lá số** | ✅ **Thỏa mãn.** Hóa Kỵ (Đ) tại **Phúc/Thân Sửu** (trục Sửu-Mùi) |
| **Mức tác động** | Cát **8/10** — Kỵ biến thành lá chắn giữ tài sản |

### 8. CỰ NHẬT ĐỒNG CUNG TẠI DẦN

| Hạng mục | Nội dung |
|---|---|
| **Tên** | Cự Nhật đồng cung tại Dần |
| **Điều kiện** | Cự Môn + Thái Dương đồng cung tại Dần, Miếu/Vượng |
| **Câu phú** | *"Cự Nhật đồng cung, quan phong tam đại"* (Quan lớn ba đời) |
| **Nguồn** | tuvichanco.net, lyso.vn, tuviglobal.com, mytour.vn |
| **Ý nghĩa** | Tài hùng biện, chí tiến thủ, sự nghiệp lớn. Phù hợp luật, ngoại giao, y sư |
| **Trên lá số** | ✅ **Thỏa mãn** tại **Điền Dần**. Cự (V) + Dương (V). Nhưng **ngộ Triệt** → Phát muộn sau 30 |
| **Mức tác động** | Cát **7/10** — Triệt giảm. Sau 30 Triệt nhả → BĐS/Hạ tầng bùng nổ |
| **Ghi chú** | *"Thạch Trung Ẩn Ngọc"* chính danh yêu cầu Cự ở Tý/Ngọ + Khoa Quyền Lộc (cohoc.vn). Cự Nhật tại Dần mang tinh thần tương tự nhưng KHÔNG phải Thạch Trung chính danh |

## II. HUNG CÁCH (Cách cục xấu/Cảnh báo)

### 1. LỘC TỒN THỦ MỆNH — KÌNH ĐÀ GIÁP

| Hạng mục | Nội dung |
|---|---|
| **Tên** | Lộc Tồn Thủ Mệnh, Kình Đà Giáp |
| **Điều kiện** | Lộc Tồn tọa thủ Mệnh → Kình Dương và Đà La TỰ ĐỘNG giáp hai bên |
| **Câu phú** | *"Lộc Tồn thủ Mệnh, Kình Đà giáp, nhược vô cát tinh, chủ cô độc"* — *"Kình Đà lưỡng giáp, nhỡ nhàng ăn xin"* |
| **Nguồn** | free.fr (Tử Vi Phú), hatvan.vn, huyenhocvietnam.vn |
| **Ý nghĩa** | Giàu nhưng cô đơn. Nếu không có cát tinh → Phá tán, bần hàn |
| **Trên lá số** | ✅ **Thỏa mãn.** Lộc tại Hợi → Kình tại Tý + Đà tại Tuất. NHƯNG có cát tinh mạnh: Phủ (Đ), Khoa (B), Tuần bảo vệ → **Hung giảm đáng kể** |
| **Mức tác động** | Hung **5/10** (giảm nhờ Phủ Khoa Tuần, so với 9/10 nếu không có cát) |
